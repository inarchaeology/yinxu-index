# 香港全真道堂科儀音樂電子數據庫

　　香港全真道堂科儀音樂電子數據庫由蓬瀛仙館設立和營運，透過影像、圖片、文字解說和樂譜等不同形式的資料，全面記錄香港全真道堂的科儀音樂。

　　數據庫收錄8個單元的科儀音樂：

- 呂祖無極寶懺
- 玄門讚星科
- 玄門攝召科（附樂譜15份）
- 玄門破獄科
- 清微禮斗科（附樂譜17份）
- 玄門開位科（附樂譜6份）
- 關燈散花科
- 三元宮傳統經韻
